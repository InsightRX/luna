# Remove all \$TABLE records from a model

Remove all \$TABLE records from a model

## Usage

``` r
remove_table_from_model(model, variables, firstonly = FALSE, file)
```

## Arguments

- model:

  pharmpy model object

- variables:

  character vector with variable names

- firstonly:

  add `FIRSTONLY` parameter to \$TABLE record

- file:

  path to file, e.g. `sdtab`
