# Get all parameters that are defined (from a predefined vector of possible parameters)

Get all parameters that are defined (from a predefined vector of
possible parameters)

## Usage

``` r
get_defined_pk_parameters(
  mod,
  possible = c("CL", "V", "V1", "V2", "V3", "Q", "Q2", "Q3", "K10", "K12", "K21", "K13",
    "K31")
)
```
