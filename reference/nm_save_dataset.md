# Save an R data.frame to a NONMEM-style dataset as CSV

Save an R data.frame to a NONMEM-style dataset as CSV

## Usage

``` r
nm_save_dataset(data, filename, tool = c("nonmem", "nlmixr"))
```

## Arguments

- data:

  data.frame

- filename:

  filename to save to.

- tool:

  type of dataset, either `nonmem` or `nlmixr`
