# Run model with pharmpy

Run model with pharmpy

## Usage

``` r
call_pharmpy_fit(
  model_file,
  path,
  clean = TRUE,
  console = TRUE,
  verbose = TRUE
)
```

## Arguments

- model_file:

  model file, e.g. "run.mod"

- path:

  run folder path, e.g. "run1"

- console:

  show output from nmfe in console? Default `FALSE`

- verbose:

  verbose output?
