# Syntax-check a NONMEM model

Syntax-check a NONMEM model

## Usage

``` r
luna_check(id, folder = NULL, verbose = FALSE, ...)
```

## Arguments

- id:

  run id, e.g. `run1`. This will be the folder in which the NONMEM model
  is run.

- folder:

  path to folder containing the model file. Default is current
  directory.
