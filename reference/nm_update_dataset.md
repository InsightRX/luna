# Update \$DATA in NONMEM model with new dataset

Update \$DATA in NONMEM model with new dataset

## Usage

``` r
nm_update_dataset(model_filename, dataset_filename, ...)
```

## Arguments

- model_filename:

  model filename

- dataset_filename:

  dataset filename

- ...:

  parameters passed to
  [`nm_save_model()`](https://insightrx.github.io/luna/reference/nm_save_model.md)
