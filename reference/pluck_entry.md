# Pluck an inner list element from an unnamed outer list, where an element matches (default name is "id") a specific value. Analogue to \_.pluck() in JS/underscore

Pluck an inner list element from an unnamed outer list, where an element
matches (default name is "id") a specific value. Analogue to \_.pluck()
in JS/underscore

## Usage

``` r
pluck_entry(x, id, el = "id")
```
