# Compare basic run info for 2 or more model runs

Compare basic run info for 2 or more model runs

## Usage

``` r
luna_compare(..., folder = ".", return_object = FALSE)
```

## Arguments

- ...:

  id of runs to compare. Can be specified either as a vector, or as
  separate arguments.

- folder:

  folder to look in

- return_object:

  return object?
