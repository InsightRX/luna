# Write a NONMEM model object to file

Write a NONMEM model object to file

## Usage

``` r
nm_save_model(model = NULL, modelfile = NULL, overwrite = FALSE)
```

## Arguments

- model:

  NONMEM model object (imported with `read_nm()`)

- modelfile:

  NONMEM model filename

- overwrite:

  overwrite model file if exists?
