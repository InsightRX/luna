# Update parameter estimates (and fix)

For example for using model in simulations.

## Usage

``` r
update_parameters(model, fit, fix = FALSE, verbose = FALSE)
```

## Arguments

- model:

  pharmpy model object or NONMEM model code (character) or path to
  NONMEM model file.

- fit:

  pharmpy fit object

- fix:

  fix the estimates?

- verbose:

  verbose output?
